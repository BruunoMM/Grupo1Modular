Para executar o programa:

1- Coloque o executavél Trab01.exe e o script TesteTurma.script na pasta de sua preferência.

2- Abra o prompt de comando do windows (cmd).

3- Utilizando o comando cd, vá até o diretório onde esta o executavél Trab01.exe e o script TesteTurma.script.
   Exemplo: cd C:\Users\Cris\Documents\Modular\Trabalho\Testes

4- Digite no prompt:
   Trab01.exe /sTesteTurma.script > saida.txt


Estruturação de diretórios:

EXECUTAVEL - Contém o arquivo .exe

FONTES - Contém o .h e .c para compilação do programa.

SCRIPTS - Contém o arquivo .script feito pelo grupo.
